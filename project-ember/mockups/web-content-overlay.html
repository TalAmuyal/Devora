Devora User Guide

Getting Started

Devora is a terminal-based IDE built for agentic development workflows. It composes existing tools — Claude Code, Kitty, and Neovim — into a focused environment for day-to-day coding. This guide covers installation, profiles, workspaces, and the core commands you will use.

To install Devora, run the following commands in your terminal: 

curl -fsSL https://raw.githubusercontent.com/TalAmuyal/Devora/master/install.sh | bash

# Verify the installation
debi health

# Launch Devora
devora
  • Each Kitty tab holds one workspace session
  • Use debi to manage workspaces and sessions from within Devora
  • Profiles let you customize settings per project or team
  • Run debi health --strict to check optional dependencies
